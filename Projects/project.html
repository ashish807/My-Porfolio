Virtual Pen

September 2020

The main feature of this project is you can write something on your screen virtually.

This Project was done using 6 steps:

  • 1) Find the color range of the target object and save it
  • 2) Apply the correct morphological operation to reduce noise in the video
  • 3) Detect and track the colored object with contour detection.
  • 3) Find the object's x,y location coordinates to draw on the screen
  • 4) Add a Wiper Functionality to wipe off the Whole screen
  • 5) Add an Eraser Functionality to erase parts of the drawing
